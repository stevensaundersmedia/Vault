# Network

One file per key contact. Use the template at `templates/network-contact.md`.

---

## How to use this folder

- Add a file for every meaningful professional contact — conferences, introductions, referrals, former colleagues worth keeping warm
- Update files after every meaningful interaction
- Before a networking conversation, ask AI to read the relevant contact file and help you prepare
- After a conversation, drop notes into `inbox/` and ask AI to update the contact file
