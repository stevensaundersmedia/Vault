# Vault — My Shared Brain

This is a vendor-agnostic AI vault built in Markdown. It acts as persistent context across every AI tool I use — Claude, ChatGPT, Gemini, or anything else.

## What this is

A structured knowledge base about me, my work, my projects, and my workflows. Instead of re-explaining context in every new AI session, I point the tool here.

## How to use it

**If you are an AI reading this:** Start with `AGENTS.md`. It contains your operating instructions, privacy rules, and a map of this vault.

**If I am setting up a new AI session:** Paste or attach `AGENTS.md` first, then the specific files relevant to the task.

## Structure

| Folder | Contents |
|---|---|
| `me/` | Identity, CV, writing voice, bio, privacy rules |
| `career/` | Career status, target roles, consultancy, network, opportunities |
| `projects/` | Active projects with context and status |
| `workflows/` | Step-by-step guides for recurring tasks |
| `fitness/` | Workout stats and training protocols |
| `inbox/` | Raw notes to be processed and filed |
| `templates/` | Reusable starting points for notes and documents |

## Principles

- Every file is plain Markdown — no tool lock-in
- Privacy rules in `me/privacy.md` apply to all AI interactions
- `career/status.md` is the most time-sensitive file — update it when things shift
- The `inbox/` folder is the capture zone; process it weekly
